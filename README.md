# SpikeScout
A management and scouting application for FRC teams. <br><br>
<strong>Built with Next.js</strong>

## Features
- Create a team for your FRC organizatiob
- Manage members, roles and permissions
- Assign tasks and issues to members
- Add events and due dates to a team-shared calendar
- Store scouting data in a database and analyse it with our tools
